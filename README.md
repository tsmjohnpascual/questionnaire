# Front End Developer Questionnaire

A questionnaire app using HTML, CSS and React.

## Getting Started

### Dependencies

* Node version v18.18.0+

### Installing

```
git clone https://github.com/tsmjohnpascual/questionnaire.git
```

### Usage

```
npm install

npm run dev
```
